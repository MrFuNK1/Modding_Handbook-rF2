###########################
Modding Handbook - rFactor2
###########################

Welcome to the in-official Modding Handbook for the racing simulation game
`rFactor2 <https://www.studio-397.com/>`_ by `Studio 397 BV <https://www.studio-397.com/about-2/>`_
, first released in 2012.

This Handbook can be used online and offline. For offline use this Handbook is
provided in HTML, PDF and EPUB formats and can be downloaded here:
